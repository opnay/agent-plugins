# Advance Codex

`advance-codex`는 Codex용 기능을 더 명확하고 안정적으로 설계하기 위한 플러그인입니다.
재사용 가능한 skill, tool-use guidance, installable plugin bundle, custom subagent 정의처럼 Codex 자체의 기능 표면을 다루는 작업을 위한 문서화와 가이드를 제공합니다.

이 플러그인은 Codex 관련 아티팩트를 더 명시적이고 유지보수 가능하게 만드는 데 목적이 있습니다.
반대로 일반적인 실행 workflow나 무관한 공용 유틸리티를 담는 용도로 넓히지 않습니다.
