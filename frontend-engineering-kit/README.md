# Frontend Engineering Kit

`frontend-engineering-kit`은 UI 코드베이스에서 더 강한 engineering decision을 내리기 위한 frontend specialist 플러그인입니다.
구조 선택, React architecture, component boundary, domain-modeling threshold, UI implementation quality, test-driven change workflow를 다루는 것이 목적입니다.

이 플러그인은 모든 문제를 generic refactoring advice로 평평하게 만들지 않고, 프론트엔드 문제를 올바른 레이어에서 다루도록 돕습니다.
반대로 프론트엔드 범위를 벗어난 execution work까지 받아들이는 catch-all 플러그인으로 넓히지 않습니다.
