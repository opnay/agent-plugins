# Workflow Kit

`workflow-kit`은 작업을 어떻게 정의하고, 실행하고, 검토하고, 마무리할지를 다루는 범용 workflow 플러그인입니다.
작업이 먼저 정의와 alignment가 필요한지, end-to-end execution이 필요한지, bounded refinement가 맞는지, blocking-first review handling이 필요한지, 마지막 readiness judgment 단계인지 구분하도록 돕습니다.

이 플러그인의 목적은 더 나은 workflow guidance로 작업의 완성도를 높이는 것입니다.
반대로 특정 도메인의 구현 가이드로 변하거나, workflow 범위를 벗어난 architecture guidance를 흡수하지 않습니다.
