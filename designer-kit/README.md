# Designer Kit

`designer-kit`은 non-coding product/interface planning workflow를 위한 design-first 플러그인입니다.
Figma나 구현으로 넘어가기 전에 design brief를 먼저 잡을지, existing direction을 critique할지, pre-code screen specification을 만들지 선택하도록 돕습니다.

이 플러그인은 design-first, pre-code 상태를 유지해야 합니다.
반대로 frontend implementation guidance나 general product execution work로 무너지지 않도록 합니다.
