# Frontend Kit

`frontend-kit`은 React-facing artifact를 `system`, `design`, `domain` 축으로 분리해서 다루기 위한 frontend 플러그인입니다.
핵심은 component, hook, utility, page, global state를 파일 종류나 익숙한 관성으로 나누는 대신, 실제 존재 이유와 change pressure를 기준으로 분류하는 것입니다.

이 플러그인은 프론트엔드 전반의 넓은 구조 선택 번들이 아니라, 축 분리와 dependency direction, 구현 순서와 정리 순서를 명시적으로 다루는 쪽에 집중합니다.
